# 技术路线的选择

同学你好！大三这个时间点其实**非常关键**，但也**完全来得及**。

很多拿到大厂Offer的同学，在大三上学期的时候状态和你是一样的。你现在的优势是：**你还有大概3-6 months的时间（寒假+大三下）来突击**。

现在的核心策略不是“全面学习”，而是**“功利性学习”**。为了找实习，你需要把所有精力集中在**一个**最容易就业、需求量最大的技术栈上，实现从“会语法”到“能干活”的转变。

以下是为你定制的**“大三突击实习”**路线图：

---

### 第一步：做减法，选定一个方向（1天）

不要试图全栈，不要试图搞懂底层原理再动手。对于初学者，目前实习岗位最多的就两个方向，请二选一：

1.  **Java 后端开发**（国内需求最大，大厂机会多，但卷，考察基础深）
2.  **Web 前端开发**（反馈快，入门相对平滑，中小厂需求也多）

*注：如果C++没有特别深厚的底子，不建议短期突击C++；Python主要用于AI/数据，本科生找开发岗不如Java容易。*

---

### 第二步：高强度技术突击（预计 1-1.5 个月）

假设你选择了最主流的 **Java 后端**（这是国内最适合应届生的方向），以下是精简版路线：

#### 1. Java 基础进阶（2周）
*   **扔掉课本**，去看培训班的视频（B站上韩顺平、尚硅谷等），倍速看。
*   **重点补：** 集合框架（ArrayList, HashMap原理，面试必问）、面向对象（接口、多态）、异常处理。
*   **了解即可：** Swing、GUI编程（完全不用看）。
*   **必须掌握：** Java 8 新特性（Lambda, Stream流），写代码会快很多。

#### 2. 数据库 SQL（1周）
*   安装 MySQL。
*   学会基本的 CRUD（增删改查）。
*   **重点：** 必须懂索引（Index）是什么，事务（Transaction）的ACID特性。这是面试题重灾区。

#### 3. 框架三件套（2-3周，核心中的核心）
*   不要学原生的 JSP/Servlet（太老了），直接学 **Spring Boot**。
*   **学习路线：** Maven（包管理） -> Spring Boot（Web开发） -> MyBatis-Plus（操作数据库）。
*   **目标：** 能写出一个只要发送 HTTP 请求，就能存取数据库数据的接口。

#### 4. 中间件（1周，加分项）
*   **Redis**：学会简单的缓存操作，面试官觉得你懂性能优化。
*   **Git**：必须会！实习第一天就要用 Git 拉代码，不会很尴尬。

---

### 第三步：项目实战（最重要的环节，1个月）

只有语法和框架知识是找不到实习的，你需要一个**“拿得出手的项目”**放在简历上。

**不要写：** 图书管理系统、学生管理系统（这些烂大街了，HR看都不看）。

**建议做的项目（B站找视频跟着敲，但要魔改）：**
1.  **外卖/电商系统**（瑞吉外卖、苍穹外卖等）：虽然也多人做，但涵盖了业务全流程。
2.  **社区/博客论坛**：加上“点赞、评论、Redis缓存热帖”功能。
3.  **秒杀系统**（高阶）：如果能把高并发秒杀讲清楚，实习稳了。

**怎么才算做好了？**
*   不是把代码敲一遍就完了。
*   要能说出**“为什么”**：为什么用 Redis？为什么要分库分表？
*   **遇到 Bug 怎么解决的**：记录下来，这是面试最好的素材。

---

### 第四步：八股文 & 算法（穿插进行，持续到面试）

你在做项目的时候，每天要留出 1-2 小时给面试题。

1.  **LeetCode（力扣）：**
    *   不要漫无目的刷。去刷 **“LeetCode 热题 100”**。
    *   重点攻克：数组、链表、二叉树、哈希表。
    *   实习面试通常不会太难，但手写“快排”、“反转链表”、“二叉树遍历”是基操。

2.  **八股文（背诵）：**
    *   去搜“Java面试高频题”或“JavaGuide”。
    *   重点背：HashMap 扩容机制、线程池参数、MySQL 索引失效场景、Spring Bean 生命周期。
    *   **理解着背**，不要死记硬背。

---

### 时间表建议（假设现在开始）

*   **第 1-3 周：** 疯狂补 Java 基础 + Spring Boot + MySQL，甚至可以不求甚解，先跑通代码。
*   **第 4-7 周：** 跟着视频做一个像样的项目，并推送到 GitHub/Gitee。
*   **第 8 周起：** 疯狂刷题（算法+八股文），同时**优化简历**。
*   **随时：** 只要有公司招实习，就开始投！**以面代练**。不要等“准备好了”再投，你永远不会觉得自己准备好了。

### 给你的心理建设

1.  **接受“烂代码”：** 刚开始写出来的代码肯定很烂，没关系，能跑通就是胜利。
2.  **善用工具：** 遇到报错不要发呆，直接把错误信息复制给 ChatGPT/DeepSeek，让它解释并给出解决方案，这是学习最快的方式。
3.  **只要一个机会：** 第一份实习不用非得是大厂，中小厂甚至创业公司都可以。只要能让你接触真实的开发流程（Git协作、上线流程、读屎山代码），就是巨大的成功。

**现在的你：**
打开 B 站/Coursera/Udemy，搜索 "Java Spring Boot 项目实战"，选一个播放量高、发布时间在 2年内的视频，**马上开始配置环境**。

动起来！祝你顺利找到实习！



## 🗺️ Java 后端实习生速成路线图 (2025版)

既然你明确选择了 **Java 后端**，这是一个非常稳妥且需求量大的方向。为了让你在大三下学期能顺利找到实习，我为你制定了一份**“速成版”Java 实习生技术路线图**。

这份路线图的原则是：**一切为了面试，一切为了能干活。** 那些工作 3-5 年才用得到的底层原理（如 JVM 调优、并发编程底层源码），暂时先放一放。

---

我将学习过程分为了四个阶段，**总耗时建议控制在 2-3 个月内**。

#### 🛠️ 第一阶段：Java 基础夯实 (耗时：2-3 周)
*目标：能看懂代码，能写基本的逻辑，理解面向对象。*

| 必须掌握的核心          | 具体知识点 (面试必问)                                        | **❌ 坚决跳过/略看**                                |
| :---------------------- | :----------------------------------------------------------- | :------------------------------------------------- |
| **基础语法**            | 变量、数据类型、循环、数组、String 类常用方法。              | 复杂的 IO 流操作、正则表达式（用的时候查）。       |
| **面向对象 (OOP)**      | **封装、继承、多态**（一定要理解概念）、接口与抽象类的区别。 | Swing、AWT、JavaFX（做桌面的，现在没人用）。       |
| **集合框架 (重中之重)** | **ArrayList vs LinkedList**、**HashMap** (底层原理必背)、HashSet。 | 老旧的 Vector、Hashtable。                         |
| **异常处理**            | try-catch-finally、自定义异常。                              |                                                    |
| **Java 8 新特性**       | **Lambda 表达式**、**Stream 流**（开发效率神器）、Optional。 |                                                    |
| **多线程基础**          | 线程创建(Thread/Runnable)、线程池基础 (ThreadPoolExecutor)。 | JUC 高并发锁源码、AQS 原理（太深了，初期先别看）。 |

*   **推荐工具**：IntelliJ IDEA (不要用 Eclipse)。
*   **学习资源**：B站搜索“韩顺平 Java”或“尚硅谷 Java 基础”，倍速观看。

---

#### 💾 第二阶段：数据库与持久层 (耗时：1-2 周)
*目标：能把数据存进数据库，能查出来。*

| 必须掌握的核心          | 具体知识点                                                   | **❌ 坚决跳过/略看**                             |
| :---------------------- | :----------------------------------------------------------- | :---------------------------------------------- |
| **MySQL 基础**          | SELECT/UPDATE/DELETE/INSERT、WHERE、GROUP BY、ORDER BY。     | 存储过程、触发器、复杂的视图。                  |
| **MySQL 进阶 (面试用)** | **索引 (Index)** 是什么、**事务 (Transaction)** 的 ACID 特性。 | 复杂的 SQL 调优、分库分表（目前了解概念即可）。 |
| **JDBC**                | 了解 Java 连接数据库的原始方式，理解 Driver、Connection。    | 手写 JDBC 工具类（太繁琐）。                    |
| **MyBatis / Plus**      | **MyBatis-Plus** (国内公司都在用，简化 CRUD)、Mapper XML 怎么写。 | Hibernate（国内用得少）。                       |

---

#### 🍃 第三阶段：Web 框架开发 (耗时：2-3 周)
*目标：能写接口，浏览器访问你的地址能返回 JSON 数据。*

这是找工作的门槛，必须熟练。

| 必须掌握的核心         | 具体知识点                                                   | **❌ 坚决跳过/略看**                                          |
| :--------------------- | :----------------------------------------------------------- | :----------------------------------------------------------- |
| **Spring 核心**        | **IOC (控制反转)**、**AOP (面向切面)** —— 只要懂概念，会背定义。 | XML 配置方式（现在都用注解）。                               |
| **Spring Boot (核心)** | 自动配置原理（了解）、Controller/Service/Dao 分层架构、RESTful 接口设计。 | Spring Cloud / Microservices（微服务架构对于实习生来说是加分项，不是必选项，先学会单体）。 |
| **Web 基础**           | HTTP 协议（GET/POST 区别）、Session/Cookie、JSON 数据格式。  | JSP、Servlet 原生开发（太老了，了解即可）。                  |
| **工具链**             | **Maven** (依赖管理)、**Git** (代码提交，实习必会)。         | Gradle（以后再学）。                                         |

---

#### 🚀 第四阶段：中间件与项目实战 (耗时：3-4 周)
*目标：做一个拿得出手的项目，模拟企业开发。*

**1. 必学中间件（加分神器）：**
*   **Redis**：做缓存用。学会 String, Hash, List 等数据结构，知道怎么在 Spring Boot 里集成 Redis。
*   *选学*：RabbitMQ / Kafka (消息队列)，如果时间不够可以先不看，或者只看概念。

**2. 核心项目实战（简历上的亮点）：**
不要自己瞎写，去 B 站找现成的项目视频跟着做，做完后**魔改**。
*   **推荐项目类型**：外卖系统（如瑞吉外卖）、电商商城（如谷粒商城-简化版）、社区论坛。
*   **项目里必须包含的技术**：Spring Boot + MyBatis-Plus + MySQL + Redis + Git。

---

### 📅 学习时间表执行建议

假设你每天能投入 4-6 小时学习：

*   **第 1-15 天：Java 基础突击**
    *   重点攻克集合（HashMap）和多线程基础。
    *   刷 LeetCode 简单题（如：两数之和、反转链表）。

*   **第 16-25 天：MySQL + MyBatis**
    *   安装好 MySQL，在 IDEA 里连接数据库。
    *   学会用 MyBatis-Plus 对一张表进行增删改查。

*   **第 26-45 天：Spring Boot + 项目实战 (重头戏)**
    *   跟着视频敲代码，**不要只看**，一定要动手敲。
    *   遇到 Bug 记录下来，这些 Bug 就是面试时最好的谈资。

*   **第 46 天起：背八股文 + 投递简历**
    *   搜“Java 面试题 2024/2025”。
    *   **以面代练**：不要怕被怼，面试失败是查漏补缺最快的方式。

### 💡 给你的特别叮嘱

1.  **关于微服务（Spring Cloud）**：
    大三实习面试，很多中小厂**不强求**你会微服务。如果你时间紧，先把 **单体架构 (Spring Boot)** 玩明白。单体都没写顺溜，去学微服务只会晕头转向。

2.  **关于算法**：
    现在的行情，算法必考。去刷 **LeetCode 热题 100**，不需要全刷完，把前 50 道高频题刷个 2-3 遍。

3.  **遇到不懂的怎么办？**
    *   **Chatgpt/DeepSeek 是最好的老师**。把错误代码粘贴进去，问它：“这段代码为什么报错？怎么修改？”
    *   不要在一个环境配置问题上卡 3 天，解决不了就重装，或者找 AI 问替代方案。

**现在，打开 IDEA，先写一个 `Hello World`，然后去 B 站搜索“Java 基础教程”，开始你的第一步吧！**
